# lab06

Лабораторная работа Лисейчиковой Екатерины

## Отчет по лабораторной работе VI

### Тема

Изучение средств пакетирования на примере CPack.

### Цель работы

Цель работы - подготовить проект к сборке установочных архивов и Linux-пакетов.

### Используемое окружение

- основной целевой сценарий: Linux;
- язык: C++11;
- система сборки: CMake;
- дополнительные инструменты добавляются по теме конкретной лабораторной;
- персональные токены и временные адреса не сохраняются в репозитории.

### Структура проекта

- `formatter_lib` - базовая статическая библиотека форматирования;
- `formatter_ex_lib` - расширенная библиотека, использующая `formatter`;
- `solver_lib` - библиотека для решения квадратного уравнения;
- `hello_world_application` - демонстрационное приложение;
- `solver_application` - приложение для вывода корней уравнения;
- `tests` - модульные тесты, начиная с lab05;
- `docs` и `scripts` - дополнительные материалы поздних лабораторных.

### Ход выполнения

- добавлены файлы `DESCRIPTION` и `ChangeLog.md`; 
- создан `CPackConfig.cmake` с метаданными пакета;
- описаны генераторы `TGZ`, `ZIP`, `DEB` и `RPM`; 
- в CMake добавлены install-правила для библиотек, headers и приложений;
- сохранена совместимость с CI и тестами предыдущих лабораторных.

### Команды для проверки

```bash
cmake -S . -B _build -DCMAKE_BUILD_TYPE=Release
cmake --build _build
cmake --install _build --prefix _install
cd _build && cpack -G TGZ
cd _build && cpack -G DEB
```

### Домашнее задание

Домашнее задание выполнено внутри этого же репозитория. Подробное описание вынесено в `HOMEWORK.md`.

Кратко по выполнению:

- пакет получает имя `formatter-workspace`; 
- для DEB задан maintainer, для RPM указаны имя, лицензия и changelog;
- описание пакета вынесено в `DESCRIPTION`; 
- установка раскладывает headers в `include`, библиотеки в `lib`, приложения в `bin`.

### Результат

Проект можно не только собрать, но и упаковать в артефакты, пригодные для передачи или публикации.

В ходе лабораторной работы закреплены практические навыки настройки проекта, автоматизации сборки и оформления результата в виде воспроизводимого репозитория. Работа подписана: Лабораторная работа Лисейчиковой Екатерины.
