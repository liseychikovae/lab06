# Домашнее задание lab06

Лабораторная работа Лисейчиковой Екатерины

## Что требовалось сделать

Домашнее задание относится к теме лабораторной работы VI: изучение средств пакетирования на примере cpack.

## Выполненные пункты

- пакет получает имя `formatter-workspace`; 
- для DEB задан maintainer, для RPM указаны имя, лицензия и changelog;
- описание пакета вынесено в `DESCRIPTION`; 
- установка раскладывает headers в `include`, библиотеки в `lib`, приложения в `bin`.

## Как проверить домашнее задание

```bash
cmake -S . -B _build -DCMAKE_BUILD_TYPE=Release
cmake --build _build
cmake --install _build --prefix _install
cd _build && cpack -G TGZ
cd _build && cpack -G DEB
```

## Пояснение по реализации

Проект оформлен так, чтобы его можно было перенести в Linux-окружение и собрать стандартными командами CMake. Файлы не завязаны на абсолютные пути macOS, персональные секреты не добавлены в git, а дополнительные инструменты подключаются через конфигурационные файлы конкретной лабораторной.
